## ✨ Возможности

### 🏠 Рабочий стол
- Лента приложений с горизонтальной прокруткой
- Часы с поддержкой **GMT** (0..+13)
- Дата с годом
- Индикатор заряда батареи в реальном времени
- Индикатор подключения к WiFi

### 📦 Приложения
| Иконка | Приложение | Описание |
|--------|------------|----------|
| ℹ️ | **Info** | Полная информация о системе, разработчике и Telegram-канал |
| 🔋 | **Bat** | Уровень заряда батареи с визуальной шкалой |
| 🐍 | **Snake** | Классическая Змейка (управление: A — поворот) |
| 🐦 | **Flappy** | Знаменитый Flappy Bird (A — прыжок, B — выход) |
| 📶 | **WiFi** | Сканер сетей + подключение с паролем (сохраняется) |
| 📊 | **Accel** | Акселерометр в реальном времени с визуализацией |
| 📰 | **News** | Новости с локального сервера (настраиваемый URL) |
| 🥚 | **Egg** | Кликер: разбивай яйца, счёт сохраняется |
| ⚡ | **Power** | Управление питанием: Перезагрузка, Выключение, Глубокий сон |
| ⚙️ | **Settings** | Яркость, GMT, News URL, О системе |
| 🎨 | **Themes** | 5 тем: Стандарт, Тёмная, Полоски, Волны, Кубы |

### 🔒 Безопасность
- Экран блокировки при включении
- Автоблокировка через **30 секунд** бездействия
- Вход по кнопкам **A** или **B**

### ⏻ Управление питанием
- **Выключение:** удержание **PWR** **2.5 секунды** (с таймером на экране)
- **Выход из приложений:** короткое нажатие **PWR** или **B**
- **Deep Sleep / Reboot:** доступны в приложении **Power**

## 🛠️ Установка

### 1. Подготовка Arduino IDE
1. Установите [Arduino IDE](https://www.arduino.cc/en/software)
2. Добавьте поддержку ESP32:
   - **Файл → Настройки → Дополнительные ссылки для Менеджера плат**
   - Вставьте: `https://espressif.github.io/arduino-esp32/package_esp32_index.json`
3. Установите плату **ESP32S3 Dev Module** через Менеджер плат

### 2. Установка библиотек
Через **Менеджер библиотек** установите:
- `M5StickCPlus2` by M5Stack

### 3. Загрузка прошивки
1. Скачайте файл `WInESP.ino` из этого репозитория
2. Откройте его в Arduino IDE
3. Подключите M5StickC Plus2 по USB-C
4. Выберите порт и плату `M5StickCPlus2`
5. Нажмите **Загрузить**

## 🎮 Управление

| Кнопка | Действие |
|--------|----------|
| **A** | Запуск приложения / Действие в игре |
| **B** (коротко) | Листать ленту / Действие в приложении |
| **PWR** (коротко) | Выход из приложения на рабочий стол |
| **PWR** (2.5 сек) | Выключение устройства |
| **PWR** (на локскрине) | Включение (быстрое) |
